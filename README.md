Studentvolgsysteem .NET MAUI

Deze applicatie is ontwikkeld in .NET MAUI en biedt één centrale omgeving waar studenten en docenten studievoortgang kunnen beheren en inzien. Het systeem combineert voortgangsregistratie, feedback en gegevensdeling in één platform.

Doel van het project

Binnen de cluster OOSDD wordt momenteel gewerkt met meerdere losse documenten zoals voortgangsdocumenten, feedbackbestanden en tijdschrijflijsten. Deze documenten zijn niet gekoppeld, waardoor docenten moeite ervaren met:

- het terugvinden van feedback  
- het verkrijgen van een compleet beeld van studentontwikkeling  

Het doel van dit project is een aanvullend studentvolgsysteem dat:

- voortgang centraliseert  
- feedback overzichtelijk weergeeft  
- koppelingen maakt tussen vaardigheden en beoordelingen  
- gegevensdeling ondersteunt waarbij de student eigenaar blijft  
- beter inzicht biedt in studieontwikkeling
